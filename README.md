# TimingFirstOrder
## 行動タイミングを推定するプログラム

## 学習コードは大きく2つ
- a_t/ .. 時定数の学習
- u_t/ .. ステップ応答の学習

## u_tの学習
  u(t) .. 非発話らしさ  
  スペクトログラムのAE ボトルネック特徴量を入力として学習  
  学習方法は2つ検討しており,  
  1). LSTM に入力する LSTM model (u_t/)  
  2). 100ms 毎に推定する Dense model (u_t_dense/)  

## a_tの学習
  a(t) .. 時定数  
  1次遅れ系のステップ応答を求める際の傾きのようなもの  
  a(t), u(t) から 発話生成らしさ y(t) を計算する  
  y(t) = a(t) * u(t) + (1 - a(t)) * y(t-1)
